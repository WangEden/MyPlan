2. [ \int_{-\infty}^{\infty} e^{-x^2} dx \ 高斯积分 \newline 但 \ e^{-x^2} \ 实数范围内不可积 ] 3.一个求导逆运算表达式（ 注意力 ） [ yy''+(y')^2=(yy')' ] 4.常系数非齐次微分方程的特解选取有一定的规律性，主要根据非齐次项的形式来选择。通常，特解的形式与方程右侧的非齐次项 (f(x)) 有关，特解的选择会保证它和方程中的非齐次项具有相同的类型。以下是一些常见的非齐次项形式及其特解的选取规律：

一般形式的常系数非齐次微分方程

常系数线性非齐次微分方程的一般形式为： [ L(y) = a_n \frac{d^n y}{dx^n} + a_{n-1} \frac{d^{n-1} y}{dx^{n-1}} + \cdots + a_1 \frac{dy}{dx} + a_0 y = f(x) ] 其中，(f(x)) 是非齐次项。特解的形式由 (f(x)) 决定，而齐次解是通过解对应的齐次方程（即 (L(y) = 0)）得到的。

特解的选取规律

根据非齐次项 (f(x)) 的形式，我们可以遵循以下规律来选取特解：

(f(x)) 是多项式的情况 ： 如果 (f(x)) 是一个多项式，例如 (f(x) = x^m)，则特解的形式可以选作一个与 (x^m) 同阶的多项式： [ y_p(x) = A_m x^m + A_{m-1} x^{m-1} + \cdots + A_1 x + A_0 ] 例如，若 (f(x) = 3x^2)，特解的形式为 (y_p(x) = A_2 x^2 + A_1 x + A_0)。

(f(x)) 是指数函数的情况 ： 如果 (f(x) = e^{\lambda x})，特解的形式通常取为： [ y_p(x) = Ae^{\lambda x} ] 例如，若 (f(x) = e^{2x})，则特解形式为 (y_p(x) = Ae^{2x})。

(f(x)) 是三角函数的情况 ： 如果 (f(x)) 是正弦或余弦函数，例如 (f(x) = \sin(\omega x)) 或 (f(x) = \cos(\omega x))，则特解通常选择为： [ y_p(x) = A \cos(\omega x) + B \sin(\omega x) ] 例如，若 (f(x) = \cos(3x))，特解形式为 (y_p(x) = A \cos(3x) + B \sin(3x))。

(f(x)) 是多项式与指数函数的乘积 ： 如果 (f(x) = x^m e^{\lambda x})，特解的形式可以选择为： [ y_p(x) = (A_m x^m + A_{m-1} x^{m-1} + \cdots + A_1 x + A_0) e^{\lambda x} ] 例如，若 (f(x) = x e^{2x})，特解形式为 (y_p(x) = (Ax + B)e^{2x})。

(f(x)) 是多项式与三角函数的乘积 ： 如果 (f(x) = x^m \cos(\omega x)) 或 (f(x) = x^m \sin(\omega x))，特解的形式通常为： [ y_p(x) = (A_m x^m + A_{m-1} x^{m-1} + \cdots + A_0) \cos(\omega x) + (B_m x^m + B_{m-1} x^{m-1} + \cdots + B_0) \sin(\omega x) ] 例如，若 (f(x) = x \sin(3x))，特解形式为： [ y_p(x) = (Ax + B) \sin(3x) + (Cx + D) \cos(3x) ]

注意：特解和齐次解的关系

如果特解的形式与齐次方程的解（齐次解）重合（即特解和齐次解有相同的形式），则特解需要乘上自变量 (x) 的适当幂次以避免重复。例如：

若非齐次项 (f(x) = e^{2x})，而齐次解中已经有项 (Ae^{2x})，则特解应选为 (y_p(x) = Bx e^{2x})，而不是 (y_p(x) = Be^{2x})。

如果齐次解中已有 (x^m) 项，则特解形式中应增加一个 (x) 作为因子，例如从 (x^m) 改为 (x^{m+1})。

总结

特解的形式通常根据非齐次项的形式选择，并确保它与齐次解不重复。常见的非齐次项形式及其对应的特解规律是求解常系数非齐次微分方程的核心步骤之一。在实际求解中，先根据非齐次项确定特解形式，然后代入原方程求解未知系数。
